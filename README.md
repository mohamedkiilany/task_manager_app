# Task Manager App

A Flutter task management app built with Clean Architecture and Riverpod. This repository includes features for projects and tasks, a simulated authentication flow, and a simple mock API integration for demo purposes.

---

## Features

- Authentication (simulated)
- Projects list with pull-to-refresh
- View, add, and complete tasks per project
- Profile screen and logout
- Consistent loading and error states

---

## Quick Start

Prerequisites:
- Flutter SDK (stable)

Commands:

```bash
git clone https://github.com/mohamedkiilany/task_manager_app.git
cd task_manager_app
flutter pub get
flutter run
```

---

## Notes

- Uses JSONPlaceholder as a mock API; writes are not persisted server-side.
- The app stores a fake JWT locally to simulate authentication.
